## LANDING PAGE

O que é uma landing page?
No bootcamp do le wagon nos ensinamos a programação da forma start up:

E normalmente o primeiro passo de uma start up é fazer um landing page pq essa start up precisa mostrar sua
proposta de valor para o mundo.
É importante tb que ela tenha um design clean - hoje em dia os padores de design sao bem elevados mas
veremos no workshop de hoje que, com ajuda de algumas bilbiotecas, nao é dificil atingir esse padrao.
A landing page de vocês tambem tem que ter um call to action - um landing page que não pdede nada ao
usuário ué uma landing page sem proposito, sem valor. é importante a interação com o usuario e nesse caso 
podemos pedir o email do usuario.

Essa interação é importante tb pra vc percerber a efetividade da sua proposta de valor -  por exemplo
se vc tem 100 visitas mas somente um usuario deixa o email, talvez seja interessante rever essa proposta de valor.

E por ultimo, é pensar no landing page como ultima etapa do seu CRM isso quer dizer que a landing page sozinha nao faz nada
mas constitui o inicio de um dialogo com o potencial usuario. Então é dessa forma q o landing ppage tem q ser pensada.
